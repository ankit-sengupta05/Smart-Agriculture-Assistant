### ![](https://img.shields.io/badge/STEP_01-BACKEND-00FF88?style=for-the-badge&logo=flask&logoColor=black) &nbsp; Flask API

```powershell
# ── Create & activate virtual environment ────────────────
python -m venv venv
venv\Scripts\activate

# ── Install dependencies ──────────────────────────────────
pip install flask flask-cors numpy pandas opencv-python scikit-learn

# ── Run API from backend/ folder ─────────────────────────
cd backend
python app.py
```

    "area_hectares"          : 10,
